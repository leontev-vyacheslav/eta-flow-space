# Plan: Replace `eval()` with safe expression evaluator

## Context

Expressions are DB-stored, admin-written, and can be complex (async `dss.getEnumDescription()` calls, boolean logic, template literals). ## Step 2 — `ast-validator.ts`

Parse expression with `acorn.parse(expr, { ecmaVersion: 2022, sourceType: 'module', allowAwaitOutsideFunction: true })`.

**Reject:** `AssignmentExpression`, `UpdateExpression`, `NewExpression`, `ImportExpression`, `ArrowFunctionExpression`, `FunctionExpression`, and `Identifier` matching `process`, `require`, `global`, `module`, `eval`, `Function`, `this`, `console`.

**Allow with validation:**
- `CallExpression` — only if callee is `MemberExpression` on whitelisted objects (`dss`)
- `MemberExpression` — only if root identifier is in `['state', 'dss', 'flowCode']`
- `BinaryExpression`, `LogicalExpression`, `UnaryExpression`, `ConditionalExpression`, `TemplateLiteral`, `AwaitExpression`, literals — recurse into children

Export: `validateAst(expression: string): void` — throws on invalid.

## Step 3 — `expression-evaluator.service.ts`

```ts
@Injectable()
export class ExpressionEvaluatorService {
  async evaluateExpression(expression: string, context: Record<string, unknown>): Promise<boolean> {
    validateAst(expression);
    const AsyncFunction = Object.getPrototypeOf(async function () {}).constructor;
    const fn = new AsyncFunction(...Object.keys(context), `return Boolean(${expression})`);
    return fn(...Object.values(context));
  }

  async evaluateDescription(description: string, context: Record<string, unknown>): Promise<string> {
    if (!description.includes('${')) return description;
    // Strip surrounding backticks if present
    const template = description.replace(/^`|`$/g, '');
    // Extract ${...} parts, evaluate each through evaluateExpression
    return template.replace(/\$\{([^}]+)\}/g, async (_, expr) => {
      const value = await this.evaluateExpression(expr.trim(), context);
      return String(value);
    });
  }
}
```

Note: `String.prototype.replace` with async replacer needs a loop-based approach (see implementation). Use a while-loop with `regex.exec()`.

## Step 6 — Tests (`expression-evaluator.service.spec.ts`)

**Valid expressions:**
- `state.isConnected === false` → true
- `state.temperature > 80` → true/false
- `state.pressure > 100 && state.temperature > 50` → combined
- `!state.isConnected` → negation
- Description: `` `Temperature is ${state.temperature}°C` `` → `"Temperature is 85°C"`
- Description: `Static text` → returned as-is

**Blocked expressions (should throw):**
- `require('fs')` → blocked
- `process.exit()` → blocked
- `eval("hack")` → blocked
- `new Function("return 1")()` → blocked
- `state.x = 5` → blocked
- `state.x++` → blocked
- `import('fs')` → blocked
